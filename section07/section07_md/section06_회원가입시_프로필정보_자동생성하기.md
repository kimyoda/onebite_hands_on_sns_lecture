## 회원가입시 프로필 정보 자동 생성하기

- supabase 관련 세팅
- type.ts에 데이터베이스 적용

회원 가입 -> 유저 정보 저장

- 트리거: 유저 정보 추가 -> profile 데이터 생성 -> profile 데이터 생성
- 예외가 발생하면 마땅히 처리할 방법이 없다.
- 처리할 방법이 없는건 아니지만 까다롭고 복잡한 과정을 거친다.
- 데이터베이스 트리거 대신 어플리케이션 기능을 이용

- 프로필 정보를 조회하고 -> 데이터가 없으면 -> 프로필을 생성하는과정
