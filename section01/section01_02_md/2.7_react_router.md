## 2.7 React_Router

### react_router

### 1. React Router란?

- React Router는 React 애플리케이션에서 **클라이언트 사이드 라우팅(Client-Side-Routing)**을 구현하기 위한 표준 라이브러리이다.
- 웹은 URL이 바뀔때마다 새로운 HTML을 받는다.
- React Router를 사용하면 **페이지 전환 시 서버 요청 없이 JavaScript가 뷰를 교체한다**
- 이것이 SPA(Single Page Applicat)이다.
  전통적 웹: URL 변경 → 서버 요청 → 새 HTML 수신 → 전체 페이지 리렌더
  React Router: URL 변경 → JS가 URL 감지 → 해당 컴포넌트 렌더링 (서버 요청 없음)
- 최신 버전은 React Router v7이고, 설치는 아래와 같다.

```bash
npm install react-router
```
