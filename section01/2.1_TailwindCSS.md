## 2.1 Tailwind CSS 소개

### Tailwind CSS 소개

Tailwind CSS는 미리 정의된 **유틸리티 클래스(Utility Class)**를 조합해 스타일링하는 CSS 프레임워크입니다.

- 유틸리티 클래스 기반 스타일링
- 클래스 조합만으로 다양한 UI 스타일 구현 가능
- 공식 문서: [https://tailwindcss.com](https://tailwindcss.com)

### 기본 사용 예시

````jsx
import "./App.css";

function App() {
  return <div className="text-white font-bold">Hello World</div>;
}

export default App;

- Tailwind CSS의 유틸리티 클래스
- 다양하고 많은 유틸리티 클래스가 많다.
- 클래스들만 잘 활용해도 모든 케이스의 스타일링이 가능

- https://tailwindCSS.com(공식문서)
- Get start with Tailwin CSS 가이드 문서에 따라 진행하면 된다

- text-white: 글자 색상을 흰색으로 지정
- font-bold: 글자 굵기를 bold로 지정

- 초기 설정 (현재 기준)
- index.css에 아래 코드를 추가합니다.

```tsx
@import "tailwindcss";
````

- 개발 편의성 도구

1. VS Code 확장 프로그램

- Tailwind CSS IntelliSense 설치

제공 기능:

- Tailwind 클래스 자동완성
- 클래스 hover 정보 표시
- 오타/유효성 체크 등

2. Prettier 플러그인 (클래스 자동 정렬)

- prettier 설정 파일(.prettierrc 등)에 아래 내용 추가:

```json
{
  "plugins": ["prettier-plugin-tailwindcss"]
}
```

- Tailwind 클래스 순서를 자동 정렬해 가독성과 유지보수성을 높일 수 있습니다.
