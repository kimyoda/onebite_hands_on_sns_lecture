본 강의는 크게 2개의 파트로 구성됩니다.

2. 한입로그 구현 파트

| 파트               | 포함 섹션   |
| ------------------ | ----------- |
| 사전 준비 파트     | 섹션 2 ~ 4  |

## React.js 실전 프로젝트 - SNS편

React.js + TypeScript + Supabase를 활용해 실무형 SNS를 구현하는 프로젝트형 강의입니다.  
기초적인 React/TypeScript 지식을 갖춘 뒤 수강하는 것을 권장합니다.

### 학습 기능 요약

| 주제      | 학습 내용                                                                           |
| --------- | ----------------------------------------------------------------------------------- |
| 인증/인가 | 이메일 로그인/회원가입, GitHub 소셜 로그인, 비밀번호 재설정, 세션 유지, 라우트 가드 |
| 피드      | `useInfiniteQuery` 기반 무한 스크롤, 로딩 상태 처리, 캐시 기반 데이터 요청 최적화   |
| 포스트    | 생성/수정/삭제, 이미지 업로드, 인가(RLS), 이탈 방지/메모리 누수 대응                |
| 좋아요    | 좋아요 처리, 상태 표시, 낙관적 업데이트, 중복 방지                                  |
| 댓글      | 댓글/대댓글 생성·수정·삭제, 무한 중첩 댓글                                          |
| 프로필    | 프로필 페이지/수정, 프로필 관련 인가 설정                                           |
| 테마      | 다크/라이트 모드 전환, 새로고침 후 테마 유지                                        |
| 기술 스택 | TailwindCSS, Zustand, TanStack Query, Supabase                                      |
