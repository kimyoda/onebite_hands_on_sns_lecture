## 2.1 Tailwind CSS 정리1

### Tailwind CSS 소개

- Tailwind CSS는 미리 정의된 **유틸리티 클래스(Utility Class)**를 조합해 스타일링하는 CSS 프레임워크이다. Bootstrap처럼 완성된 컴포넌트를 제공하는 것이 아닌, 작은 단위의 클래스들을 자유롭게 조합해 원하는 디자인을 만들 수 있다.

* 유틸리티 클래스 기반 스타일링
* 클래스 조합만으로 다양한 UI 스타일 구현 가능
* 별도의 CSS 파일 없이 HTML(JSX)안에서 스타일 완성
* 공식 문서: [https://tailwindcss.com](https://tailwindcss.com)

### 기본 개념

- 유틸리티 클래스(Utility Class)
- 유틸리티 크래스는 하나의 CSS 속성에 대응하는 작고 단일한 목적의 클래스이다.

```css
/* Tailwind가 내부적으로 생성하는 스타일 예시 */
.text-white {
  color: #ffffff;
}
.font-bold {
  font-weight: 700;
}
.bg-blue-500 {
  background-color: #3b82f6;
}
.p-4 {
  padding: 1rem;
}
.mt-2 {
  margin-top: 0.5rem;
}
```

- Tailwind 방식:

```jsx
<div className="p-4 bg-white rounded-lg shadow-md">카드</div>
```

- 초기 설정

- 설치

```bash
npm install tailwindcss @tailwindcss/vite
```

- index.css 추가

```css
@import "tailwindcss";
```

> 현재기준 Tailwind CSS v4부터 별도의 `tailwind.config.js` 설정 없이 CSS 파일에 `@import "tailwindcss"`만 추가해도 동작한다.

### 📝 텍스트 (Typography)

| 클래스                  | 설명           | CSS 동일                                                         |
| :---------------------- | :------------- | :--------------------------------------------------------------- |
| `text-xs`               | 아주 작은 글자 | `font-size: 0.75rem`                                             |
| `text-sm`               | 작은 글자      | `font-size: 0.875rem`                                            |
| `text-base`             | 기본 글자      | `font-size: 1rem`                                                |
| `text-lg`               | 큰 글자        | `font-size: 1.125rem`                                            |
| `text-xl`               | 더 큰 글자     | `font-size: 1.25rem`                                             |
| `text-2xl` ~ `text-9xl` | 초대형 글자    | `font-size: 1.5rem ~ 8rem`                                       |
| `font-thin`             | 얇은 굵기      | `font-weight: 100`                                               |
| `font-normal`           | 기본 굵기      | `font-weight: 400`                                               |
| `font-semibold`         | 세미볼드       | `font-weight: 600`                                               |
| `font-bold`             | 볼드           | `font-weight: 700`                                               |
| `text-white`            | 흰색 글자      | `color: #fff`                                                    |
| `text-gray-500`         | 회색 글자      | `color: #6b7280`                                                 |
| `text-blue-600`         | 파란 글자      | `color: #2563eb`                                                 |
| `text-center`           | 가운데 정렬    | `text-align: center`                                             |
| `text-left`             | 왼쪽 정렬      | `text-align: left`                                               |
| `uppercase`             | 대문자 변환    | `text-transform: uppercase`                                      |
| `underline`             | 밑줄           | `text-decoration: underline`                                     |
| `truncate`              | 말줄임표 처리  | `overflow: hidden; white-space: nowrap; text-overflow: ellipsis` |

### 📐 여백 (Spacing)

> **참고:** Tailwind의 간격은 **4px = 1 단위** 기준이다. (예: `p-4` = 16px)

| 클래스      | 설명                           |
| :---------- | :----------------------------- |
| `p-4`       | 모든 방향 padding 16px         |
| `px-4`      | 좌우 padding 16px              |
| `py-2`      | 상하 padding 8px               |
| `pt-2`      | 위 padding 8px                 |
| `pb-4`      | 아래 padding 16px              |
| `m-4`       | 모든 방향 margin 16px          |
| `mx-auto`   | 좌우 margin auto (가운데 정렬) |
| `mt-8`      | 위 margin 32px                 |
| `gap-4`     | flex/grid 간격 16px            |
| `space-x-4` | 자식 요소 좌우 간격            |
| `space-y-2` | 자식 요소 상하 간격            |

### 🎨 색상 (Colors)

Tailwind는 기본적으로 풍부한 색상 팔레트를 제공하며, `{속성}-{색상}-{농도}` 형식을 사용한다.

| 구분          | 값                                                                                                                                                      |
| :------------ | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **농도 단계** | 50, 100, 200, 300, 400, 500, 600, 700, 800, 900, 950                                                                                                    |
| **색상 계열** | slate, gray, zinc, neutral, stone, red, orange, amber, yellow, lime, green, emerald, teal, cyan, sky, blue, indigo, violet, purple, fuchsia, pink, rose |

### 개발 편의성 도구

- 1. VS Code 확장 프로그램 - Tailwind CSS IntelliSense
     설치 후 제공되는 기능:
  - Tailwind 클래스 자동완성(입력 시 클래스 목록 표시)
  - 클래스 hover시 실제 CSS 미리보기
  - 오타 및 유효하지 않은 클래스 경고
  - `@apply` 사용 시 동작

- 2. Prettier 플러그인 (클래스 자동 정렬)
     설치 :

```bash
npm install -D prettier prettier-plugin-tailwindcss
```

`.prettierrc` 설정:

```json
{
  "plugins": ["prettier-plugin-tailwindcss"]
}
```

- Tailwind 공식 권장 순서, 클래스를 자동 정렬해 가독성과 유지보수성을 높인다.

### ⚖️ Tailwind CSS vs 기존 CSS 비교

| 항목             | 기존 CSS/CSS Module   | Tailwind CSS                        |
| :--------------- | :-------------------- | :---------------------------------- |
| **스타일 위치**  | 별도 CSS 파일         | JSX 클래스 직접                     |
| **네이밍**       | 클래스명 직접 작성    | 미리 정의된 클래스 사용             |
| **번들 크기**    | 사용하지 않아도 포함  | 사용한 클래스만 포함 (Tree-shaking) |
| **커스터마이징** | 자유롭지만 규칙 필요  | 디자인 시스템 기반                  |
| **반응형**       | 미디어 쿼리 직접 작성 | `md:`, `lg:` 접두사로 간편 처리     |
| **러닝커브**     | 낮음                  | 초반 클래스 학습 필요               |
